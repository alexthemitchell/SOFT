# The Simple Online Functional Teacher (SOFT) Language
Introduction to SOFT: http://soft-lang.org/TeachSOFT.pdf

## To Install
1. Install `ghc`
2. Clone this repo
3. Edit paths.txt to reflect your file system

## To Make and Run
  `make && ./soft`
## To Make Only
  `make`

## To Interpret a file written in SOFT
  ` ./soft <file>`
